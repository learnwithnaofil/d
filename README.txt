How to use:
1. Open index.html in your browser.
2. Replace placeholder images in assets/images/ with your original institution images.
3. Replace logo placeholders with actual logos using the same filenames:
   - school-logo.png
   - degree-college-logo.png
   - university-of-mumbai-logo.png
4. Replace phone numbers, email addresses and address with verified official details.
5. Replace placeholder PDFs in assets/documents/ with approved official forms and prospectus.
6. Upload all files to GitHub Pages, Hostinger, cPanel, Netlify or any static hosting.

Important content guidance:
- Do not publish fake NAAC grades, fake rankings or unverified achievements.
- Add University notices only after verifying source and date.
- Keep school and degree college identities separate in notices and downloads.
- For forms, connect static forms later with Google Forms, EmailJS, Formspree or a backend.

Customization:
- Main colors are defined in assets/css/style.css under :root variables.
- Navigation is repeated inside all HTML files because this is a pure static website.
- To change menu items, update the header section in all pages or use your editor's global find-and-replace.

Suggested next improvements:
- Add original campus photographs.

- Add downloadable official admission form and prospectus.
- Add Google Map iframe on contact.html.
- Connect enquiry forms using Google Forms or EmailJS.
